LeetCode189 Rotate Array

2017-02-03 | LeetCode | LeetCode, 数组

描述

Rotate an array of n elements to the right by k steps.

样例

思路

将数组中每个元素循环右移k位。

不妨记A = [1, 2, 3, 4]，B = [5, 6, 7]，其目标就是是将原数组从 AB 变成 BA，所以可以运用$BA = (A^TB^T)^T$这一性质。注：$A^T$的意思是将数组$A$中元素翻转。
